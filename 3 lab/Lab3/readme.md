На многопроцессорной системе потоки работают истинно параллельно - каждый на своем ядре

На одном процессоре потоки работают псевдопараллельно - планировщик ОС создает иллюзию параллелизма за счет быстрого переключения

Производительность значительно снижается при ограничении до одного процессора из-за накладных расходов на переключение контекста

Графики выполнения демонстрируют эффективность работы планировщика ОС Windows в организации многозадачной работы потоков
